# One-step-to-V2ray #

一个一键安装/配置V2ray的脚本

## 获取&使用 ##

```
wget https://github.com/johnpoint/One-step-to-V2ray/raw/master/v2ray.sh && chmod +x v2ray && ./v2ray
```

## 功能 ##

- 生成服务端与客户端
- 支持动态端口
